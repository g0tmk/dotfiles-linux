#### how to build xmobar (info only)
- `sudo aptitude install cabal-install`
- `sudo aptitude install libxpm-dev libasound2-dev libghc-libxml-sax-dev c2hs libiw-dev`
- `cabal update`
- `cabal install xmobar --flags="all_extensions"`
